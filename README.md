## td (ToDo) - fast and simple CLI task manager

> [!WARNING]
> Highly beta

`td` works using sqlite3 database, so the one requirement is sqlite3.

### Requirements

- sqlite3
- make(to actually bulid project)
- 
### Quick start
Dowload the repo and run 
```
make
```
to build `td`. 
### Getting help
Just run `td help`.
